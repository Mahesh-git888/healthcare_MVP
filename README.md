# Healthcare Referral System MVP

Full-stack healthcare referral system with a Next.js frontend, NestJS backend, PostgreSQL partner/admin storage, and Google Sheets lead logging.

## Folder structure

```text
.
|-- backend
|   |-- src
|   |   |-- admin
|   |   |-- auth
|   |   |-- common
|   |   |-- database
|   |   |-- google-sheets
|   |   |-- lead
|   |   `-- partner
|   |-- package.json
|   `-- tsconfig.json
|-- frontend
|   |-- app
|   |-- components
|   |-- lib
|   |-- package.json
|   `-- tailwind.config.ts
|-- docker-compose.yml
|-- package.json
`-- render.yaml
```

## Stack

- Frontend: Next.js 16, React 19, TypeScript, Tailwind CSS
- Backend: NestJS 11, TypeORM, PostgreSQL
- Lead storage: Google Sheets API
- Frontend deployment: Vercel
- Backend deployment: Render

## Local setup

1. Start PostgreSQL:

   ```bash
   docker compose up -d
   ```

2. Install workspace dependencies:

   ```bash
   npm install
   ```

3. Copy env files:

   - `backend/.env.example` -> `backend/.env`
   - `frontend/.env.example` -> `frontend/.env.local`

4. Run the backend:

   ```bash
   npm run dev:backend
   ```

5. Run the frontend:

   ```bash
   npm run dev:frontend
   ```

## Google Sheets columns

The backend appends rows in this order:

1. Timestamp
2. Partner Name
3. Partner Phone
4. Partner Role
5. City
6. Area
7. Patient Name
8. Patient Phone
9. Service Type
10. Shift Type
11. Status
12. Notes
13. BDO ID

## Deployment

### Vercel

- Import the `frontend` directory as a Next.js project.
- Set `NEXT_PUBLIC_API_URL` to the deployed Render backend URL.

### Render

- Import this repository and point the service to `backend`, or use the included `render.yaml`.
- Set all required backend environment variables.
- Render will expose `process.env.PORT` automatically.

## Notes

- Admin accounts are seeded automatically from `ADMIN_EMAIL` and `ADMIN_PASSWORD` if they do not already exist.
- Access links are generated by the admin dashboard and exchanged for a partner session token on `/access`.
- Leads are not stored in PostgreSQL in this MVP; they are appended directly to Google Sheets.
